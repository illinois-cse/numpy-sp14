A Short Course for Engineers

Python is one of the most popular programming languages today. NumPy, SciPy, and their affiliated ecosystem are used for advanced data processing and numerical computation for engineering applications. This short course is designed to introduce you to the fundamentals of the language and syntax through engineering exercises and physical models.

This course is not an introduction to programming, nor is it intended to replace courses such as CS 101 which introduce programming concepts. This course is strongly oriented towards practical tasks which engineering students and researchers may encounter and does not cover the fundamentals of Python.

Examples and applications will be of broad interest to engineers, and will attempt to exemplify good practices for general problem solving. Our time constraints will not permit the in-depth motivation of the numerical methods selected for inclusion. We recommend taking a numerical methods course or reading a good book on the subject if you would like the details and criteria of applicability of many of these methods. (We warmly recommend Acton or Press et al. if you are in search of a text.)

Setup

EWS Workstation: Python is installed but is not up-to-date.

  • Open a command-line window.

  • Enter module load canopy. Enter canopy.

  • Click on "Package Manager". Click on "Updates". Update scipy and ipython.

  • Close Canopy when you are done.

  • Test it by opening a command-line window and entering ipython. If IPython loads, type exit().

Own Machine, don't know if Python is installed:

  • Install Enthought Canopy Distribution, which is free for academic users. (Make sure to get this license if you can.)

  • Test it by opening a command-line window and entering ipython. If IPython loads, type exit().

Own Machine, Python already installed: Presumably you know what you're doing with Python. In this case, just be aware that we use recent versions of numpy, scipy, matplotlib, pandas, mpmath, and timeit. We may even get to pyflakes or pep8 and autopep8 if there's time.

  • Test it by opening a command-line window and entering ipython. If IPython loads, type exit().

Everyone: Once Python and IPython are installed and working, follow these commands to be prepared for the workshop.

  • Download the notebook to your Desktop folder.

  • Open a new command-line window and enter cd Desktop.

  • Input the following at the command line: ipython notebook material.ipynb &.
